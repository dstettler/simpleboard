About the platform

Designed to feel modern, minimal, and competitive

This project aims to create a chess platform where players can log in, manage games, view their activity, and eventually play in a fully interactive environment. The design focuses on readability, structure, and a premium visual feel.

With Angular on the frontend and Go on the backend, the platform is being built to support both clean UI development and scalable game logic.
