# Simpleboard ♟️

**Simpleboard** is a no-frills online multiplayer chess game where users can jump straight into a game **with or without an account**

- Users **may login/register** to track games and identity or play instantly without any signup 
- The game logic runs entirely on the server side ensuring fair play reducing the risk of client side cheating

## Features
- Real-time multiplayer chess
- Optional login / registration
- Instant play without authentication
- Public matchmaking queue
- Server side game validation
